## Quick reference

- API base: `/api/v1`; auth: Bearer JWT (access) + rotating refresh; 401 →
  auto-refresh on the client.
- Video: backend-issued Daily.co tokens only; private rooms `kalba-{workshop_id}`;
  participant-minute budget guard keeps usage inside Daily's free tier.
- Visibility: workshops are group-membership-gated (non-members get 404).
- Time: stored UTC-naive, served as `Z`-suffixed ISO.
