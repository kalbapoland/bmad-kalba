# Kalba — Project Overview

> Generated by BMAD `document-project` (deep scan) · 2026-06-27

## What Kalba is

Kalba is a **mobile-first meditation & workshop platform**. Trainers create
**groups** and schedule **workshops** (live, time-boxed sessions); members
subscribe to groups, enroll in workshops, and join **live video calls**
(Daily.co) at the scheduled time. A personal **"My Kalba"** dashboard tracks
each user's monthly practice goal and progress, and **push notifications**
remind people before a workshop starts.

The product is near its **App Store launch** (iOS bundle `com.kalba.app`,
ASC App ID `6761315112`).

## Tech stack at a glance

**Backend** — Python 3.13, FastAPI (async), SQLModel (SQLAlchemy 2.0 +
Pydantic), PostgreSQL via `asyncpg`, Alembic migrations, PyJWT + passlib/bcrypt
auth, httpx for outbound calls (Google tokeninfo, Daily.co, Expo Push, Brevo).
Packaged with `uv`, containerized, deployed on **Fly.io** (`backend-kalba`,
region `ams`), migrations run on boot.

**Frontend** — React 19 + React Native 0.81, Expo SDK 54 (new architecture
enabled), Expo Router (file-based), NativeWind v4 (Tailwind v3), Zustand (auth
store), TanStack Query v5 (server state), Axios (HTTP with refresh-token
interceptor), `@daily-co/react-native-daily-js` (WebRTC video), expo-auth-session
(Google), expo-secure-store (tokens), expo-notifications (push), i18next
(localization). Built/distributed via **EAS**.

## Core domains

1. **Auth** — Google ID-token exchange *and* native email/password, JWT access
   + rotating refresh tokens, password reset (Brevo email + server-rendered web
   page), account deletion (App Store requirement).
2. **Groups** — trainer-owned, soft-deletable; members subscribe; visibility of
   workshops is gated by group membership.
3. **Workshops** — trainer-created within a group; scheduling, capacity,
   timezone, price, hashtags (`#tag`), enrollment with race-safe capacity,
   reschedule notifications, reminders.
4. **Video** — backend-centric Daily.co integration: backend creates private
   rooms, issues short-lived meeting tokens, enforces rules, and guards a
   **participant-minute budget** so usage never leaves Daily's free tier.
5. **My Kalba** — per-user monthly goal, weekly/monthly completion stats, and
   in-app notifications.
6. **Notifications** — Expo push tokens (device-keyed), a DB-polling scheduler
   that sends workshop reminders, plus in-app notifications.
